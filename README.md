# Loan Approval Prediction System – ER Diagram

## 📌 Description

This project presents the Entity-Relationship (ER) diagram for a Loan Approval Prediction System. It illustrates how different entities such as users, datasets, machine learning models, and prediction outputs interact within the system.

## 🧩 Entities

* User
* Country
* HDI Input Data
* ML Model
* HDI Prediction
* Dataset
* Visualization Report
* Session

## 🔑 Relationships

* One User can submit multiple inputs (1:M)
* One Country can have multiple input records (1:M)
* One Input generates one Prediction (1:1)
* One ML Model generates many Predictions (1:M)
* One Prediction creates many Reports (1:M)
* One Dataset trains many Models (1:M)
* One User has many Sessions (1:M)
